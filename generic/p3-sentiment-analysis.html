Project ID: Sentiment Analysis on Tweets

Duration

1 month (2024)

Industry

Financial Markets / Technology

Dataset

9,000+ labeled training tweets + 2,000+ unlabeled test tweets

Collaboration

Team Work

Skillset

Tensorflow/Keras, Hugging Face, Text Preprocessing, Transformers, NLTK/VADER

Data Source

Public - Twitter

Project Scope

Classify financial tweets into bullish, bearish, or neutral sentiments using NLP techniques. For the classification task, we assesed traditional ML modes, deep learning (LSTM) and transformer models (DistilBERT, GPT-2, FinBERT). The workflow included data exploration, preprocessing (tokenization, cleaning) and model evaluation.

Key Challenges

  • Handling unstructured and noisy text data
  • Highly imbalanced classes - neutral tweets dominated, requiring class weighting to avoid bias
  • Designing preprocessing pipelines tailored to the strengths and limitations of each model family

Technical Approach

Phase Techniques
Data Preparation Lowercasing, URL/ticker removal, stopword filtering (retaining negations), emoji extraction, stemming/lemmatization
Feature Engineering Bag-of-Words and TF-IDF vectorization
Model Selection DistilBERT
Evaluation F1-score macro, Precision, Confusion matrix

Results

87% Accuracy

F1-Score macro of 83%

Key Observations and Future Improvements

  • Adopting the latest GPT model version and fine-tuning it for domain-specific data
  • Training FinBERT on the dataset instead of relying solely on pre-trained weights
  • Applying grid search (or other optimization techniques) to identify optimal hyperparameters for the best-performing model.
