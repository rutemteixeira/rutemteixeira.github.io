3 weeks (2025)

Industry

Retail, Wholesale Cash&Carry / B2B platform

Dataset

Products: 4,500+ rows, 3 columns
Transactions: 884,000+ rows, 3 columns

Collaboration

Team Work

Skillset

Python, RapidFuzz, Dice Similarity, Monte carlo simulation

Data Source

Internal from Recheio within Jerónimo Martins Group (Confidential)

Project Scope

The project aims to enhance customer segmentation and recommendation capabilities for retail clients. We addressed 89.49% missing client type data to build four distinct customer profiles. Based on these profiles, we designed three recommendation systems—Smart Basket, Did You Forget?, and Brand Substitution/Upselling—to improve product relevance. The expected outcome is to increase sales conversion and strengthen customer loyalty.

Key Challenges

  • Massive Data Gap: Imputing the missing ID Client Type was the foundational challenge, requiring a robust modelling approach
  • Designing and implementing different recommendation strategies that cater to each client history
  • Computational cost of Monte carlo simulation and ensuring reproducible method throught team and models used

Technical Approach

Phase Techniques
Data Preparation Merging transaction, product, and client data;label encoding; handling class imbalance with SMOTE.
Feature Engineering Client-Product category purchase matrices; used ZIP code as a location proxy
Model Selection Imputation: Ensemble model (Soft Voting with Random Forest, XGBoost, Extra Trees).

Recommenders: Dice similarity for co-occurrence
Evaluation Imputation:F1-score macro, Accuracy.

Reccommender: Accuracy via Monte Carlo simulations

Results

63% Accuracy and F1-Score

Imputation of Client ID

60% Monte Carlo Accuracy

"Did You Forget?" - Top-3 recommendations

44% Monte Carlo Accuracy

"Smart Basket" - Personalized Top-10 recommendations

Key Observations and Future Improvements

  • The "Smart Basket" personalized model (43.5% accuracy) vastly outperformed the generic group-based model (8.7% accuracy)
  • Integrate real-time stock levels and product pricing data to enable out-of-stock substitution and price-range filtering
