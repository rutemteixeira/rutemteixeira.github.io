Injury Claim Severity Classification

Duration

3 months (2023)

Dataset

15,000 claims, 85 features

Tools

Python, XGBoost, Pandas

Project Goal

Developed a machine learning system to automate severity classification of workplace injury claims, reducing manual assessment time from 3 days to 2 hours while maintaining 87% accuracy.

Key Challenges

  • Highly imbalanced classes (only 5% high severity cases)
  • Missing data in critical injury description fields
  • 90% categorical features requiring special encoding

Technical Approach

Phase Techniques
Data Preparation SMOTE oversampling, Target Encoding
Model Selection XGBoost with custom class weights
Evaluation Precision-Recall focus, SHAP analysis

Results & Impact

87% Accuracy

Outperformed manual assessment consistency by 22%

52% Faster

Reduced processing time from 72hrs to 2hrs
